🚗 Interactive Car Price Prediction – End-To-End Data Science Playground
Interactive Car Price Prediction is a Streamlit-based interactive web application that automates the entire data science workflow for the Car Price Prediction dataset. It simplifies automotive data analysis by combining data understanding, cleaning, filtering, visualization, and machine learning prediction into a single intuitive interface.
Features
📂 Fixed Dataset Input
Pre-loaded Car Price Prediction dataset (2,500 Records · 10 Features)
🔍 Automatic Data Detection
Identifies numeric and categorical columns automatically
📊 Data Understanding
Dataset shape, missing values, duplicates, data types, descriptive statistics
🧹 Data Cleaning
Removes duplicates, fills missing values (Mean/Mode), IQR outlier removal, cleaning log
🎯 Data Filtering
Filters by Brand, Fuel Type, Transmission, Condition, Year and Price range
📈 EDA
Statistical summaries, correlation matrix, categorical value counts
📉 Visualizations
Heatmap, Histogram, Scatter plot, Line chart, Bar chart, Box plot with auto-generated insights
🤖 ML Modelling
Regression → Linear, Ridge, Random Forest, Gradient Boosting
Classification → Logistic Regression (Budget vs Premium)
Metrics: R², RMSE, MAE, Accuracy, Precision, Recall, F1-Score, Confusion Matrix
